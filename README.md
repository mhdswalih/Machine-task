# Project Title

Short project description.

## Structure
- `/Backend` — Node + Express + TypeScript backend
- `/Frontend` — React (Vite) frontend

## Quick start (backend)
1. cd Backend
2. npm install
3. npm run dev

## Quick start (frontend)
1. cd Frontend
2. npm install
3. npm run dev

## Deploy / Submit
Push repository to GitHub with `/Backend` and `/Frontend` at the repo root.
